## Local development

### Shadcn

The UI is created by using [shadcn](https://ui.shadcn.com/).

To add a component use the _shadcn CLI_. For example, to add the button component:

```bash
pnpm shadcn add button
```

## Services

### Spotify

To enable the personalized search of events for the user, the login with the Spotify account option must be configured. The web app must be properly registered on https://developer.spotify.com/.

This web app uses the Spotify [Web API](https://developer.spotify.com/documentation/web-api).
